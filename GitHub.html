Nano Exercices !

Exercice 1 :

Créer un dossier exerciceNano dans le dossier utilisateur

mkdir exerciceNano

Exercice 2 :

Créer un fichier dans le dossier créé dans l'exercice 1 qui sera nommé index.html

cd exercice Nano / touch index html

Exercice 3 :

Ouvrir le fichier avec nano et écrire dedans le Doctype html puis ouvrir dans firefox

nano index.html / firefox index.html

Exercice 4 :

Ajouter dans le fichier index.html la balise title et la remplir

<title>*****</title>

Exercice 5 :

Ajouter 3 titres h2 et les remplir

<h2>****</h2> x3

Exercice 6 :

En se servant des options de nano remplacer les balises h2 par des paragraphes

ctrl\ remplacer h2 par p
